# Services Documentation

## GitHub Services

**Description**: Automate GitHub notifications and repository management. Get updates on issues, pull requests, and repository stars, and take automated actions when certain conditions are met.

**Actions**:
- **Push Event**: Triggers when a new commit is pushed to a repository.
- **Star Event**: Triggers when someone stars your repository.

**Reactions**:
- **Create Issue**: Automatically create a new issue in a repository.

**How to Use**:
1. Go to the **GitHub Services** section in the app.
2. Authenticate with GitHub to link your repositories.
3. Set up the triggers and actions you want to automate for your projects.
